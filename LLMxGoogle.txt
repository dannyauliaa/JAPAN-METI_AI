LLM x Google Apps Script

<Objective>
Create a system that sends an automatic reply email based on the input from a Google 
Form. Use GEMINI API to generate the reply.

<Purpose>
To test your skills in connecting Google Apps Script with an LLM API (the free Gemini API).

<Requirements>
1. Tools and APIs 
    
Google Account: Use your personal Google Account. NOTE: school or company organization accounts are NOT  allowed, because the app settings are different from normal ones. 
Google Apps Script:  Use Google Apps Script.
Free Gemini API:  Use the free Gemini API as the LLM API.
